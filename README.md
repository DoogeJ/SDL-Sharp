is a SDL2 for c#
The wrapper provides bindings for the following libraries:
- SDL2 (2.0.14)
- SDL2_image (2.0.5)
- SDL2_mixer (2.0.4)
- SDL2_ttf (2.0.15)
and contains the necessary DLLs for the project
